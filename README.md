# Embedded Systems & Electronics Practice

A structured repository for learning Electronics, Embedded Systems, IoT, and hardware-software integration progressively.

---

# Purpose

This repository is used to:

- Learn electronics fundamentals
- Practice embedded programming
- Build ESP32 and Arduino projects
- Understand hardware-software interaction
- Improve debugging skills
- Document practical experiments
- Develop engineering habits
